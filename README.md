## ğŸ¯ About the Project

**Postpartum Pal** is a web application that provides women with postpartum specialist advice, social connections, and essential resources for post-delivery care. The platform addresses the critical gap in postpartum support by offering:

- Evidence-based information and tutorials
- Direct communication with healthcare professionals
- Peer support through mother-to-mother chat
- Immunization tracking and reminders
- Appointment booking with specialists
- AI chatbot system 

The application serves three primary user roles: **Mothers**, **Doctors (Admin)**, and **Super Admin**.
